The *\CDB\MModel\* folder _SHALL_ be the root directory of the MModel library which _SHALL_ be composed of the following datasets.

1.  MModelGeometry

2.  MModelDescriptor

3.  MModelTexture

4.  MModelMaterial

5.  MModelCMT

6.  MModelSignature
